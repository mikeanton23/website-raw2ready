Statistics Module

Exploratory statistical analysis of fermentation datasets.

Overview

The Statistics Module performs exploratory data analysis (EDA) directly on the harmonized datasets stored in Runtime Memory.

It provides descriptive statistics, missing-value analysis, correlation analysis, outlier detection and distribution analysis that assist researchers in understanding fermentation experiments before applying advanced modelling or machine learning techniques.

All statistical analyses are performed automatically on the active dataset without requiring external statistical software.

Objectives

  • Summarize numerical variables.
  • Evaluate data quality.
  • Identify missing values.
  • Detect outliers.
  • Study variable distributions.
  • Measure variable correlations.
  • Support AI-based interpretation.

Missing Value Analysis

Before performing advanced statistical analyses, Raw2Ready automatically evaluates the completeness of the active dataset.

Missing values are identified for every variable, allowing researchers to assess data quality and determine whether preprocessing is required.

  • Missing values per column
  • Missing value percentage
  • Complete observations
  • Missing-value visualization

Distribution Analysis

Understanding the distribution of experimental variables is essential for selecting appropriate statistical methods and detecting abnormal measurements.

Raw2Ready generates distribution summaries for every numerical variable stored in Runtime Memory.

Histograms

Histograms illustrate how observations are distributed across value ranges and help identify skewness, multimodal behaviour and irregular data distributions.

  • Normal distributions
  • Right-skewed variables
  • Left-skewed variables
  • Multimodal distributions

Boxplots

Boxplots summarize the spread of each variable while highlighting quartiles, medians and possible outliers.

  • Median
  • Interquartile range
  • Minimum and maximum
  • Potential outliers

Correlation Analysis

Correlation analysis measures the strength of linear relationships between fermentation variables.

Strong positive or negative correlations often reveal biologically meaningful interactions between process variables.

Correlation Heatmaps

Correlation matrices are visualized using heatmaps, allowing rapid identification of highly correlated variables.

Correlation Interpretation
+1.00 Perfect positive relationship
+0.70 Strong positive relationship
0.00 No linear relationship
-0.70 Strong negative relationship
-1.00 Perfect negative relationship

Scatter Matrix

Scatter matrices compare every pair of variables simultaneously, revealing hidden dependencies and nonlinear behaviour.

Outlier Detection

Outlier analysis automatically identifies observations that deviate substantially from the remaining dataset.

  • Extreme sensor values
  • Instrument failures
  • Experimental anomalies
  • Unexpected biological events

Normality Assessment

Many statistical methods assume normally distributed variables.

Raw2Ready evaluates distribution characteristics to assist researchers in selecting appropriate statistical analyses.

  • Histogram inspection
  • Boxplot inspection
  • Distribution symmetry
  • Skewness evaluation

Best Practices

  • Review missing values before analysis.
  • Inspect distributions prior to hypothesis testing.
  • Investigate outliers rather than removing them automatically.
  • Interpret correlations together with process knowledge.
  • Compare statistical summaries across experiments.
  • Combine visualizations with numerical statistics.
  • Validate AI-generated interpretations using expert knowledge.

Troubleshooting

Problem Possible Cause Recommended Solution
No statistics generated No dataset loaded Load a dataset before opening the Statistics Module.
Unexpected summary values Incorrect preprocessing Review the preprocessing and merge steps.
Missing variables Columns removed earlier Verify the active Runtime Memory dataset.
Correlation unavailable Insufficient numerical variables Select datasets containing numerical measurements.
Outliers not detected Uniform distribution Inspect boxplots and variable distributions manually.

Summary

The Statistics Module provides a comprehensive exploratory analysis environment for fermentation datasets by combining descriptive statistics, distribution analysis, correlation analysis, missing-value assessment and outlier detection.

Integrated with Runtime Memory and the Multi-Agent AI framework, statistical outputs become directly available for automated interpretation and evidence-based reasoning.

Statistical analysis is the foundation for understanding experimental behaviour before applying predictive modelling or machine learning.

Next Step

After completing statistical analysis, users can continue with the Metadata Module to annotate experiments using the Minimum Information Framework for Fermentation Experiments (MIFE) and enrich datasets with structured experimental context.
