Raw2Ready Modules & Features

Complete reference of the core modules and capabilities available in Raw2Ready.

Core Processing Modules

Module Description Input Output
Load Import raw fermentation datasets. CSV, XLSX DataFrame
Merge Synchronize heterogeneous time-series. Multiple datasets Merged dataset
Calculate Generate derived process variables. Formula definitions Extended dataset
Plot Create interactive visualizations. Dataset columns Publication-ready plots
Metadata MIFE annotation and protocol management. Experimental metadata Structured metadata

Multi-Agent Framework

Raw2Ready integrates a collaborative multi-agent architecture that combines experimental datasets, metadata, BacDive knowledge, scientific literature and external web resources.

Available Agents

  • Agent Orchestrator
  • Data Analyst
  • BacDive Explorer
  • Crossref Explorer
  • Internet Explorer
  • Metadata Analyst
  • Literature Reviewer
  • Master Summarizer

Agent Capabilities

Example Commands


# Parse raw files

python raw2ready.py parse \
-cf examples/config.json

# Merge datasets

python raw2ready.py merge \
-cf examples/config.json

# Calculate new variables

python raw2ready.py calculate \
-cf examples/config.json

# Launch NiceGUI

python run_app.py
