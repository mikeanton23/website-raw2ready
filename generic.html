Raw2Ready Documentation

Complete user guide for industrial biotechnology data harmonization and AI-assisted exploration.

Overview

Raw2Ready is an open-source framework for the harmonization, preprocessing, visualization, semantic annotation and intelligent exploration of industrial biotechnology datasets.

The framework transforms heterogeneous experimental data originating from multiple fermentation devices into standardized, machine-readable and AI-ready datasets.

What to Anticipate

  • Streamline fermentation data preprocessing.
  • Synchronize asynchronous time-series.
  • Generate publication-quality visualizations.
  • Perform statistical analyses automatically.
  • Annotate experiments using metadata standards.
  • Interact with experiments using AI agents.

Installation

Requirements

  • Python 3.10 or newer
  • Linux, Windows or macOS
  • Optional: Ollama for local LLM execution

git clone https://gitlab.com/bioindustry4.0/raw2ready.git

cd raw2ready

python3 -m venv raw2ready

source raw2ready/bin/activate

pip install -r requirements.txt

Launch NiceGUI


python run_app.py

Open your browser: 


http://localhost:8080

Basic Workflow

  1. Load raw experimental data.
  2. Clean and harmonize datasets.
  3. Merge multiple instruments.
  4. Calculate derived variables.
  5. Visualize experimental trends.
  6. Annotate metadata.
  7. Interact with AI agents.

Load Data

The Load module imports heterogeneous datasets produced by industrial biotechnology equipment.

Supported Formats

  • CSV
  • Excel (.xlsx)
  • Sartorius exports
  • BioLectorXT exports

Features

  • Dataset preview
  • Column inspection
  • Missing-value detection
  • Automatic datatype identification

Visualization

Raw2Ready provides interactive and publication-quality visualizations.

Available Plots

  • Line plots
  • Scatter plots
  • Histograms
  • Box plots
  • Bar charts
  • Multi-series plots

Typical use cases include biomass growth, oxygen limitation and process monitoring.

Statistical Analysis

The Statistical Framework automatically characterizes fermentation datasets.

Implemented Analyses

  • Descriptive statistics
  • Missing value analysis
  • Distribution analysis
  • Correlation analysis
  • Outlier detection
  • Normality testing

Metadata Annotation

Raw2Ready supports metadata annotation using the Minimum Information Framework for Fermentation Experiments (MIFE).

Metadata Categories

  • Organism information
  • Protocol descriptions
  • Growth conditions
  • Experimental settings
  • Sampling information

Multi-Agent AI Framework

Raw2Ready integrates a collaborative multi-agent architecture capable of reasoning over datasets, metadata, microbial repositories and scientific literature.

Available Agents

  • Agent Orchestrator
  • Data Analyst Agent
  • BacDive Explorer
  • Crossref Explorer
  • Internet Explorer
  • Literature Reviewer
  • Metadata Analyst
  • Master Summarizer

The agents cooperate to provide evidence-based responses for industrial biotechnology experiments.
