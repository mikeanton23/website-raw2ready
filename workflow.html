Raw2Ready Workflow

Complete end-to-end workflow for industrial biotechnology datasets.

Overview

Raw2Ready transforms heterogeneous fermentation datasets into standardized, semantically enriched and AI-ready knowledge.

Workflow Pipeline

Raw Files → Load → Harmonize → Merge → Calculate → Statistics & Visualization → Metadata (MIFE) → Multi-Agent AI → Export

Step 1 — Load

Import Sartorius, BioLectorXT, gas analyzer, CSV and Excel datasets.

Step 2 — Harmonize

Standardize names, units, timestamps and missing values.

Step 3 — Merge

Synchronize heterogeneous time-series using timestamp alignment.

Step 4 — Calculate

Create derived variables such as OUR, CER and growth-rate metrics.

Step 5 — Statistics & Visualization

Generate descriptive statistics and publication-quality figures.

Step 6 — Metadata

Annotate experiments using the MIFE metadata framework.

Step 7 — Multi-Agent AI

  • Agent Orchestrator
  • Data Analyst
  • Metadata Analyst
  • BacDive Explorer
  • Crossref Explorer
  • Internet Explorer
  • Literature Reviewer
  • Master Summarizer

Step 8 — Export

Export harmonized datasets, figures, logs and AI-generated reports.
